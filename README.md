# PocketDhamma

PocketDhamma is a lightweight, static mobile app built with Flutter for offline reading of the _Dhammapada_.

## Features

- Offline access to multiple Dhammapada translations
- Clean, minimalistic user interface
- Adaptive light and dark mode toggle with Material You 3 design
- Modern, dynamic theming for seamless platform integration
- Cross-platform support (Android & iOS)

## Installation

Clone the repo and build the app with Flutter:

```bash
git clone https://github.com/s4nj1th/pocket-dhamma
cd pocket-dhamma
flutter pub get

# Build for Android
flutter build apk

# Build for iOS
flutter build ios
```

Install the generated app on your device.
